# School-Attendance-system

A modern, user-friendly web application designed to help teachers efficiently manage and track student attendance in schools.

## Features

- **Teacher Login Portal** - Secure login system for teachers to access the attendance management system
- **Attendance Tracking** - Easily mark students as present or absent for each class
- **Multiple Classes** - Manage attendance across different classes (10-A, 10-B, 9-A, etc.)
- **Attendance Statistics** - View real-time attendance counts (Total, Present, Absent)
- **Subject Selection** - Track attendance by different subjects
- **Attendance History** - Access and review past attendance records
- **Lesson Planner** - Plan and organize lessons with subject information
- **Student Management** - Add and manage student information
- **Mark All Feature** - Quickly mark all students as present or absent
- **Export Options** - Print or export attendance data
- **Responsive Design** - Works seamlessly on desktop and tablet devices

## How to Use

1. **Open the Application**
   - Open `index.html` in a web browser to launch the application

2. **Login**
   - Enter a username and password to access the teacher portal
   - The system will validate your credentials and grant access

3. **Select a Class**
   - Choose a class from the dropdown menu on the dashboard
   - View the list of students in that class

4. **Mark Attendance**
   - Click on individual students to toggle their attendance status
   - Use the "Mark All Present" button to quickly mark all students as present
   - Click the "Save Attendance" button to save the attendance record

5. **Switch Tabs**
   - Use the **Attendance** tab for daily attendance tracking
   - Use the **Lesson Planner** tab to plan and organize lessons

6. **View History**
   - Click the history icon (clock) in the header to view past attendance records
   - Review attendance statistics for previous dates
